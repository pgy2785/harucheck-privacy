# 개인정보처리방침

**하루체크** (이하 "앱")는 사용자의 개인정보를 소중히 보호합니다.

## 1. 개인정보의 수집 및 이용 목적

앱은 다음의 목적을 위해 최소한의 개인정보를 수집합니다.

- 할 일 관리 및 실행 기록 저장
- 클라우드 동기화를 위한 계정 식별 (Google 로그인 사용 시)
- 구독 상태 확인 및 관리
- 앱 사용성 개선 및 오류 분석

## 2. 수집하는 개인정보 항목

- Google 로그인 시: 이메일 주소, 표시 이름, 프로필 사진 URL
- 자동 수집: 기기 정보 (OS 버전, 앱 버전), 광고 ID (광고 표시 시)

로그인 없이 사용하는 경우, 모든 데이터는 기기 내에만 저장되며 외부로 전송되지 않습니다.

## 3. 개인정보의 보유 및 이용 기간

- 회원 탈퇴 또는 계정 삭제 요청 시 즉시 파기합니다.
- 로컬 데이터는 앱 삭제 시 자동으로 삭제됩니다.
- 클라우드 데이터는 계정 삭제 요청 후 30일 이내에 완전히 삭제됩니다.

## 4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우 예외로 합니다.

- 사용자가 사전에 동의한 경우
- 법령에 의해 요구되는 경우

## 5. 광고

앱은 Google AdMob을 통해 배너 광고를 표시할 수 있습니다.

- 광고 제공을 위해 광고 ID가 수집될 수 있습니다.
- 하루체크 플러스 구독 시 모든 광고가 제거됩니다.
- 광고 관련 데이터 처리는 Google의 개인정보처리방침을 따릅니다.

## 6. 데이터 저장 및 보안

- 로컬 데이터: 기기 내부 저장소에 저장됩니다.
- 클라우드 데이터: Google Firebase Firestore에 저장되며, 전송 시 SSL/TLS 암호화가 적용됩니다.
- 백업 파일: 사용자가 직접 내보낸 파일은 사용자의 책임 하에 관리됩니다.

## 7. 사용자의 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다.

- 개인정보 열람, 수정, 삭제 요청
- 계정 탈퇴 및 데이터 삭제 요청
- 마케팅 목적의 개인정보 처리 거부

요청은 아래 연락처로 보내주시면 신속히 처리하겠습니다.

## 8. 개인정보 처리방침의 변경

본 방침은 법령 변경이나 서비스 변경에 따라 수정될 수 있으며, 변경 시 앱 내 공지를 통해 안내합니다.
